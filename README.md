# ⚡ VoltWise

Sistema full-stack para gerenciamento de veículos elétricos, controle de recargas, simulador de autonomia e mapeamento de pontos de carregamento.

## 🚀 Tecnologias Cadastradas (MVP)

- **Frontend:** React, Vite, React Router, Context API
- **Backend:** Node.js, Express, Prisma ORM
- **Banco de Dados:** MySql
